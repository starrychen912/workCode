# Vue 3 + Vite

## 前言
`cloudwms-front` 项目是 `B端项目` 
## 简介
`cloudwms-front` 使用了最新的`vue3`、`vite2` 等主流技术开发，开箱即用的中后台前端解决方案。

项目目的：

- 学习`Vue3`相关Api
- 掌握`Vite2`插件机制、构建配置

## 安装使用

- 获取代码

```sh
git clone https://gitee.com/ddtechinfo/cloudwms-front.git
```

- 安装依赖

```sh
npm install
```

- 运行

```sh
npm serve
```

- 打包

```sh
npm build
```

- 本地预览
```sh
npm dev
```
## 浏览器支持
